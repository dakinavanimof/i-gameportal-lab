# i-gameportal-lab

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态页面资源的集合与备份。

## 项目简介

i-gameportal-lab 旨在收集、整理和托管一系列独立的 HTML 页面。这些页面可能包含工具、小游戏、演示、文档或其他前端内容，每个页面均可独立运行，不依赖外部服务或特定后端。

本仓库适合：
- 归档实验性或临时性的 HTML 页面
- 提供可直接访问的静态页面副本
- 分享不涉及具体业务或域名的通用前端资源

## 目录说明

仓库根目录下包含以下内容：

- `pages/` — 存放所有归档的 HTML 页面文件，每个页面为一个独立的 `.html` 文件或子目录。
- `assets/` — 页面可能用到的公共资源，如样式表、脚本、图片等（按需组织）。
- `index.html` — 可选的仓库首页，用于导航或列出所有页面。
- `README.md` — 本文件。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，包含必要的 `<head>` 和 `<body>` 结构。
- 页面内引用的资源（如 CSS、JS、图片）优先使用相对路径，确保在本地或静态托管中正常工作。
- 归档时请保持页面独立性，避免依赖外部链接或在线服务。
- 若页面包含敏感信息或测试数据，请事先清理或脱敏。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已有内容。
- 欢迎通过 Issues 提交建议或报告问题，但请注意不要包含具体域名或推广信息。
- 提交 Pull Request 时，请将新页面放入 `pages/` 目录，并确保路径正确、内容自包含。
- 仓库不承担页面内容准确性的责任，所有页面仅供学习、测试或存档用途。
